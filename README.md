# 🚗 AI-Like Car Recommendation & Comparison App

## 📌 Overview
This is a full-stack web application that allows users to compare cars and receive smart recommendations based on their preferences.

The system simulates AI behavior using a custom rule-based scoring algorithm — no external AI APIs required.

---

## 🚀 Features

- 🔍 Search cars by brand or model  
- 🎚️ Filter by price, year, and fuel type  
- ⚖️ Compare multiple cars side-by-side  
- 🧠 Smart recommendation system (AI-like logic)  
- 💬 Dynamic explanation generator  
- ❤️ Favorite cars (saved locally)  
- 📱 Fully responsive modern UI  

---

## 🧠 How It Works

Users enter:
- Budget 💰  
- Fuel type ⛽  
- Usage (city / long-distance / mixed) 🚗  

The system:
1. Filters cars  
2. Scores them based on multiple factors  
3. Selects top matches  
4. Generates human-like explanations
